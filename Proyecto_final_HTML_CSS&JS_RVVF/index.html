Breve historia de Linux

Como antecedente a la creación de linux se debe mencionar a Ken Thompson y Dennis Ritchie que fueron los creadores del sistema operativo UNIX que en un principio se llamó UNICS. AT&T puso a UNIX a disposición de universidades y compañías, incluyendo al gobierno de EUA a través de licencias, producto de esto surgieron varias versiones. En 1983 Richard Stallman, buscó desarrollar un sistema operativo que fuera de acceso libre y desarrolló el proyecto GNU para crear el un sistema similar a UNIX, el sistema debía desarrollarse sobre un núcleo que en un principo se llamó Alix y que después el programador Michael Bushnell renombró Hurd pero éste no tuvo éxito.

Finalmente en 1991 Linus Torvalds comenzó a trabajar sobre el código fuente del sistema Minix que desarrolló Andrew Tenembaum y que más tarde llegó a ser el núcleo de Linux . De esta manera surge GNU/Linux

Después de la creación de GNU/Linux en 1991 y durante aproximadamente 11 años para hacer el mantenimiento del código se utilizaban parches, hasta el 2002 que utilizó por poco tiempo un software para el control de versiones llamado BitKeeper. Fue en el año 2005 que surgió Git el cual fue desarrollado por Linus Torvalds.

Git

Git es una herramienta que realiza una función del control de versiones de código de forma distribuida. Tiene las siguientes características:

    Fue diseñada por Linus Torvalds.
    No depende de un repositorio central.
    Es software libre.
    Permite mantener un historial completo de versiones.
    Podemos movernos, como si tuviéramos un puntero en el tiempo, por todas las revisiones de código y desplazarnos una manera muy ágil.
    Es muy rápida.
    Tiene un sistema de trabajo con ramas que lo hace especialmente potente.
    Las ramas pueden tener una línea de progreso diferente de la rama principal donde está el core de nuestro desarrollo. En algún momento podemos llegar a probar algunas de esas mejoras o cambios en el código y hacer una fusión a nuestro proyecto principal, ya que todo esto lo maneja Git de una forma muy eficiente.

GitHub

Github es una plataforma de desarrollo colaborativo para alojar proyectos, utiliza el sistema de control de versiones Git. Entre algunas de sus caracteristicas se encuentran:

  • Fue desarrollado por Chris Wanstrath, P. J. Hyett, Tom Preston-Werner y Scott Chacon.

  • El software que opera GitHub fue escrito en Ruby on Rails.

  • Github permite gestionar las aplicaciones que utilizan el sistema Git. Permite la creación de repositorios de forma gratuita.

  • Muchos consideran a Github como una red social para desarrolladores que permite conectarlos para colaborar en la mejora de algún proyecto, de manera que se pueden dejar comentarios sobre el código o para reportar errores.

  • Fue comprada por Microsoft en junio del 2018.
